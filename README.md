# Calculator-CLI-App
A simple Command-Line Interface (CLI) Calculator built using Python. This application allows users to perform basic arithmetic operations directly from the terminal.

# Features
1. Addition (+)
2. Subtraction (−)
3. Multiplication (×)
4. Division (÷)
5. Continuous calculation until user exits
6. Input validation and error handling
7. User-friendly CLI interface


# Concepts Used
- Functions
- Loops
- Conditionals
- Exception Handling
